# Digital Odyssey

> Un jurnal-interfață bilingv care cartografiază codul, structura și învățarea recursivă printr-un sistem web cinematic.

English version: [README.md](./README.md)

## Prezentare generală

Digital Odyssey este un portofoliu bilingv de tip edutainment și o platformă de documentare, construit pe trei piloni: **tehnic** (cod curat, conform standardelor), **pedagogic** (explicații auto-documentate și trasee structurate de învățare) și **narativ** (o metaforă cinematică de navigație cosmică ce transformă interfața într-un spațiu de povestire).

Reunește studiul frontend-ului, sistemele vizuale, arhitectura semantică, fundamentele backend, bazele de date, principiile UX și gândirea recursivă într-un singur spațiu de proiect navigabil.

În loc să afișeze doar rezultate finisate, proiectul păstrează și procesul: jurnale, experimente structurale, studii de interfață și iterații tehnice.

## Concept

Proiectul este gândit ca mai mult decât un portofoliu standard.  
Funcționează ca o arhivă vie a explorării, în care codul, cercetarea, designul de interfață și reflecția tehnică sunt tratate ca părți ale aceluiași sistem.

Limbajul său vizual se inspiră din console de comandă, structuri HUD, blueprint-uri recursive și o logică de navigație cinematică, transformând documentația într-o parte a experienței. Designul este inspirat de filosofia explorării, astronomie, astrofizică, literatură, artă, fotografie, muzică, cinema și operele care conturează tonul contemplativ și tehnic al proiectului — cu `Solaris` ca referință atmosferică centrală.

## Obiective principale

- Construirea unui spațiu de proiect bilingv în engleză și română.
- Documentarea învățării tehnice ca parcurs vizibil și structurat.
- Combinarea clarității educaționale cu o identitate vizuală puternică.
- Explorarea arhitecturii frontend, backend, baze de date, UX și deployment prin sisteme modulare și recursive de design.
- Păstrarea atât a execuției, cât și a reflecției în interiorul aceleiași platforme.

## Aria actuală

În prezent, proiectul explorează:

- HTML semantic și arhitectură structurată a conținutului cu template-uri riguroase pentru head;
- sisteme CSS cinematice, logică de layout responsiv, estetică CRT și design HUD cu glassmorphism;
- comportament JavaScript interactiv, navigare bazată pe scroll, urmărire prin Intersection Observer și animații de mașină de scris;
- fundamente backend: arhitectură de bază, API-uri, autentificare, middleware, testare, performanță și deployment;
- paradigme de baze de date relaționale (SQL) și non-relaționale (NoSQL);
- JSON ca punte de schimb de date între frontend și sistemele de date;
- fundamente UX și accesibilitate web (WCAG, ARIA);
- fluxuri de lucru pentru controlul versiunilor (Git, GitHub) și concepte de pipeline CI/CD;
- concepte de interfață recursive și design auto-documentant;
- organizarea conținutului bilingv pe straturi paralele ale proiectului.

## Configurare locală

Clonează repository-ul:

```bash
git clone https://github.com/iuliacadar/digital-odyssey.git
cd digital-odyssey
```

Apoi deschide punctul de intrare în engleză sau română local, cu un server de development precum Live Server în VS Code.
