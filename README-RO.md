# Digital Odyssey

> Un jurnal-interfață bilingv care cartografiază codul, structura și învățarea recursivă printr-un sistem web cinematic.

English version: [README.md](./README.md)

## Prezentare generală

Digital Odyssey este un portofoliu bilingv de tip edutainment și o platformă de documentare.  
Reunește studiul frontend-ului, sistemele vizuale, arhitectura semantică și gândirea recursivă într-un singur spațiu de proiect navigabil.

În loc să afișeze doar rezultate finisate, proiectul păstrează și procesul: jurnale, experimente structurale, studii de interfață și iterații tehnice.

## Concept

Proiectul este gândit ca mai mult decât un portofoliu standard.  
Funcționează ca o arhivă vie a explorării, în care codul, cercetarea, designul de interfață și reflecția tehnică sunt tratate ca părți ale aceluiași sistem.

Limbajul său vizual se inspiră din console de comandă, structuri HUD, blueprint-uri recursive și o logică de navigație cinematică, transformând documentația într-o parte a experienței.

## Obiective principale

- Construirea unui spațiu de proiect bilingv în engleză și română.
- Documentarea învățării tehnice ca parcurs vizibil și structurat.
- Combinarea clarității educaționale cu o identitate vizuală puternică.
- Explorarea arhitecturii frontend prin sisteme modulare și recursive de design.
- Păstrarea atât a execuției, cât și a reflecției în interiorul aceleiași platforme.

## Aria actuală

În prezent, proiectul explorează:

- HTML semantic și arhitectură structurată a conținutului;
- sisteme CSS cinematice și logică de layout responsiv;
- comportament JavaScript interactiv și feedback de navigație;
- concepte de interfață recursive și design auto-documentant;
- organizarea conținutului bilingv pe straturi paralele ale proiectului.

## Configurare locală

Clonează repository-ul:

```bash
git clone https://github.com/iuliacadar/digital-odyssey.git
cd digital-odyssey
```

Apoi deschide punctul de intrare în engleză sau română local, cu un server de development precum Live Server în VS Code.

## Status

Digital Odyssey se află în prezent într-o etapă activă de construcție și refactorizare.  
Repository-ul este modelat ca un sistem bilingv de lungă durată, care îmbină logica de portofoliu, documentația tehnică și designul experimental de interfață.
