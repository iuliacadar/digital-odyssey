# Digital Odyssey

> A bilingual interface-journal mapping code, structure, and recursive learning through a cinematic web system.

Romanian version: [README-RO.md](./README-RO.md)

RO: *Digital Odyssey* este un portofoliu bilingv, experimental, construit la intersecția dintre documentație tehnică, jurnal de învățare și design de interfață cinematic.

## Overview

Digital Odyssey is a bilingual edutainment portfolio and documentation platform.  
It brings together frontend study, visual systems, semantic architecture, and recursive thinking inside a single navigable project space.

Rather than displaying only polished outcomes, the project also preserves process: logs, structural experiments, interface studies, and technical iterations.

## Concept

The project is designed as more than a standard portfolio.  
It functions as a living archive of exploration, where code, research, interface design, and technical reflection are treated as parts of the same system.

Its visual language draws from command consoles, HUD structures, recursive blueprints, and cinematic navigation logic, turning documentation into part of the experience itself.

## Core Goals

- Build a bilingual project space in English and Romanian.
- Document technical learning as a visible and structured journey.
- Combine educational clarity with a strong visual identity.
- Explore frontend architecture through modular, recursive design systems.
- Preserve both execution and reflection inside the same platform.

## Local Setup

Clone the repository:

```bash
git clone https://github.com/iuliacadar/digital-odyssey.git
cd digital-odyssey
```

Then open the English or Romanian entry point locally with a development server such as Live Server in VS Code.
